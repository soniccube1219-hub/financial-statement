# Cases Directory

This directory contains various financial analysis cases. Each case is structured to help users understand and analyze financial statements effectively.

## Structure

- **case-1-company-valuation**: This case includes methods and analyses for valuing a company based on its financial statements.
- **case-2-investment-analysis**: This case explores different investment opportunities and analyzes their potential returns based on financial metrics.
- **case-3-budget-forecast**: This case focuses on forecasting budgets and understanding financial planning through historical data.

### Adding New Cases

To add a new case:
1. Create a new folder in the `cases` directory.
2. Add `financial_statements.xlsx`, `analysis.md`, and `summary.md` files to the new folder.
3. Follow the structure used in existing cases for consistency.
4. Document any specific details or reasoning for your case in the `summary.md` file.
